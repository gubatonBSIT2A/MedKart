About the application

The MedKart application is a shop application for medical equipment. The Goal of this application is to give accessiblty to shop for medical items in a mobile phone. This would give a users time to get an item if they are away from any medical shop in the area. We the Tech Dudes hope that it would be a great help to our medical professionals to have this application for there needs
